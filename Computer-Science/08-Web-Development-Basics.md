# Web Development Basics

Web development is the process of building websites and web applications.
It is divided into three main parts: Frontend, Backend, and Full-Stack.

---

## 1. What is Web Development?
Web development includes all work needed to create:
- Static websites
- Dynamic web apps
- E-commerce platforms
- APIs
- Cloud services

---

## 2. Three Main Parts

### Frontend (Client-side)
What the user sees and interacts with.
- HTML: Structure
- CSS: Style
- JavaScript: Behavior

### Backend (Server-side)
Behind-the-scenes logic.
- Server
- Database
- APIs
- Business logic

### Full-Stack
Combines Frontend + Backend.

---

## 3. Core Frontend Technologies

### HTML
HyperText Markup Language  
Defines the **structure** of a webpage.

```html
<h1>Hello World</h1>
<p>This is a paragraph.</p>
```

### CSS
Cascading Style Sheets  
Defines the **design and layout**.

```css
h1 {
    color: blue;
    font-size: 24px;
}
```

### JavaScript
Adds **interactivity**.

```javascript
console.log("Hello Web");
```

### Popular Frontend Frameworks
- React
- Vue
- Angular
- Svelte

## 6. HTTP / HTTPS

### HTTP
HyperText Transfer Protocol  
Rules for sending data over the web.

### HTTPS
Secure version of HTTP.
- Encrypted
- Uses SSL/TLS

### Common HTTP Methods
- GET: Read data
- POST: Submit data
- PUT: Update data
- DELETE: Remove data

### Common Status Codes
- 200: OK
- 301: Redirect
- 400: Bad Request
- 401: Unauthorized
- 403: Forbidden
- 404: Not Found
- 500: Server Error

---

## 7. API (Application Programming Interface)
Allows different software to communicate.

### REST API
Most common type:
- Uses HTTP methods
- Stateless
- Returns JSON/XML

### GraphQL
Alternative to REST:
- Ask for exactly what you need
- One endpoint
